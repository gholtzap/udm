# UDM

**This repo is not production-ready yet. I am still developing the core features.**

The UDM (Unified Data Management) is a core component of the 5G architecture. It is responsible for storing subscriber data including profiles, authentication keys, and subscription information.

`udm-typescript` is a TypeScript implementation of 3GPP's 5G UDM specification. The specification can be found [here](https://portal.3gpp.org/desktopmodules/Specifications/SpecificationDetails.aspx?specificationId=3342).
